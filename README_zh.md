## 简介

这是一本实战手册——写给每一个想让AI Agent团队真正跑起来的人。从选Agent到搭团队，从建制度到防翻车，每章都是方法论+真实案例+可操作建议。

你将学会如何从零搭建一支完整运作的Agent团队：写出真正管用的System Prompt，设计可扩展的分工体系，构建记忆与沟通协议，打造越用越聪明的自进化系统。
